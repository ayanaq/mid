1. создаем новую папку для проекта
2. открываем терминал в этой папке или командную строку
3. в терминале выполняем команду git init , чтобы создать новый репозиторий
4. кладем в папку файлы нашего проекта, затем выполняем git add . для добавления всех файлов в репозиторий
5. фиксируем изменения командной git commit -m “”
6. создаем новый репозиторий на гитхаб и копируем его url
7. возвращаемся в терминал и выполняем git remote add origin <url>, чтобы связать наш локальный репозиторий с с репозиторием на гитхаб
8. отправляем файлы на гитхаб с помощью git push -u origin master
9. наш проект доступен на гитхаб
